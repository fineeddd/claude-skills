# 📚 Systematic Learning — Claude Code Skill

一个专为 Claude Code 设计的**系统化课程与书籍学习教练** Skill。

> English version: [systematic-learning-en](../systematic-learning-en/)

提供文件路径，Claude 自动读取内容，引导你完成从画像→路线图→逐章学习→费曼验证→闭环的完整学习旅程，并将输出保存为本地 Markdown 文件。

---

## 功能特点

- **读取本地文件**：支持 PDF、TXT、MD 或整个目录，无需知识库
- **自适应课程类型**：自动判断实操类 / 认知类 / 思辨类，调整提问策略
- **学习者画像**：3-5 轮对话了解背景、动机、能力水位
- **精确路线图**：分阶段生成，精确到每个章节，带 🔴/🟡/⚪ 学习深度标记
- **逐章领学**：全景拆解 + 逐知识块讲解（讲解→连接→确认），结合你的个人背景
- **费曼验证**：让你主动讲，Claude 追问，找出"以为懂但其实没懂"的缺口
- **阶段闭环**：掌握情况盘点 + 闭环决策（重学/补练/进阶）
- **自动存档**：路线图、学习笔记、验证记录保存为本地 MD 文件

## 安装方法

### 方式一：手动安装

```bash
mkdir -p ~/.claude/skills/systematic-learning
curl -o ~/.claude/skills/systematic-learning/SKILL.md \
  https://raw.githubusercontent.com/fineeddd/claude-skills/main/systematic-learning/SKILL.md
```

### 方式二：克隆仓库

```bash
git clone https://github.com/fineeddd/claude-skills.git
cp -r claude-skills/systematic-learning ~/.claude/skills/
```

## 使用方法

在 Claude Code 中，直接告诉 Claude 你想系统学习某个课程，并提供文件路径：

```
帮我系统学习 /path/to/your/course
```

**支持的触发方式：**
- `帮我系统学习 [文件路径]`
- `我想系统学这本书：[文件路径]`
- `用学习框架帮我学 [文件路径]`
- `按步骤学习这门课 [文件路径]`

**支持的文件格式：**
- 单个文件：`.txt`、`.md`、`.pdf`
- 目录：包含多个课程文件的文件夹

## 学习流程

```
启动 → Claude 读取文件
       ↓
第一步  了解你（3-5 轮对话）→ 输出学习者画像
       ↓
第二步  课程结构分析 → 作者逻辑线 + 目标对应
       ↓
第三步  生成学习路线图 → 分阶段，精确到章节
       ↓
  ┌─── 学习循环（每章重复）─────────────┐
  │  4a 章节全景拆解                    │
  │  4b 逐知识块学习                    │
  │  4c 章节收尾总结                    │
  │  5b 费曼验证（日常）                │
  │  阶段结束 → 5a 阶段性验证           │
  │  第六步  回顾与闭环                 │
  └─────────────────────────────────────┘
```

**生成的文件（保存在课程目录下的 `学习记录/`）：**

| 文件 | 内容 |
|------|------|
| `01_学习者画像.md` | 认知地图、核心动机、学习偏好 |
| `02_课程结构分析.md` | 价值判断、模块依赖、完整性检查 |
| `03_学习路线图.md` | 分阶段路线图（随进度更新掌握状态）|
| `04_学习笔记.md` | 逐章节笔记（追加写入）|
| `05_验证记录.md` | 费曼验证和阶段测试记录 |
